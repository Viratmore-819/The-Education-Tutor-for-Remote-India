# The-Education-Tutor-for-Remote-India

Apna Tuition - Connect with Expert Home Tutors
A comprehensive tuition management platform connecting students with qualified home tutors and online teachers across Pakistan.

![Live Demo](https://apna-tuition.com) ![TypeScript](https://www.typescriptlang.org/) ![React](https://reactjs.org/) ![Supabase](https://supabase.io/)

Features
For Students & Parents
Request Tuition - Submit tuition requirements easily
Browse Qualified Tutors - View tutor profiles with qualifications and experience
Direct Contact - Connect with tutors via WhatsApp
Responsive Design - Works seamlessly on all devices
For Tutors
Professional Profiles - Create detailed tutor profiles
Browse Opportunities - Find tuition jobs matching your expertise
Application Management - Track tuition applications
Approval System - Admin-verified tutor profiles
For Admins
Admin Dashboard - Manage tutors, tuitions, and requests
Tutor Approval - Review and approve tutor applications
Analytics - Track platform statistics
Content Management - Manage blog posts and FAQs
Tech Stack
Frontend: React 18 + TypeScript + Vite
Styling: Tailwind CSS + shadcn/ui
Backend: Supabase (PostgreSQL + Auth + Edge Functions)
Routing: React Router v6
Icons: Lucide React
Deployment: DigitalOcean App Platform
Installation
Clone the repository
   git clone https://github.com/ShababAkbar/Tuition-App.git
   cd Tuition-App
Install dependencies
   npm install
Set up environment variables
   cp .env.example .env
Add your Supabase credentials:

VITE\_SUPABASE\_URL=your\_supabase\_url
VITE\_SUPABASE\_ANON\_KEY=your\_supabase\_anon\_key
Run database migrations

Import migrations from supabase/migrations/ to your Supabase project
Start development server

npm run dev
Build for production

npm run build
